# Sample Library

## Install Dependencies

### Python
```bash
source .venv/bin/activate
```

## Run Project

### Run The Database
```bash
docker compose up
```

### Run the Frontend:
```bash
npm run dev
```

### Run the Backend:
```bash
cd api
fastapi dev main.py
```

## View DB with MongoExpress
```
http://localhost:8081/
```
